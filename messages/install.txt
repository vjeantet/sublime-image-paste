MarkdownImagePaste
==================

Paste a clipboard image straight into a markdown file.

In a markdown view, press Cmd+V (macOS): if the clipboard holds an image, it is
saved next to the current .md file and a ![](path) reference is inserted.
Otherwise the normal paste happens.

Requirements
------------
- macOS only (ships a universal arm64 + x86_64 helper binary).
- Sublime Text 4 (the package runs under the Python 3.8 plugin host).

Configuration
-------------
Preferences > Package Settings > Markdown Image Paste > Settings

    {
        // "" = same folder as the .md. Example: "assets".
        "image_subdir": ""
    }

Source & issues: https://github.com/vjeantet/sublime-image-paste
