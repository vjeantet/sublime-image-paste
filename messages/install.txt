MarkdownImagePaste
==================

Paste a clipboard image straight into a markdown file.

In a markdown view, press the paste shortcut (Cmd+V on macOS, Ctrl+V on
Windows): if the clipboard holds an image, it is saved next to the current .md
file and a ![](path) reference is inserted. Otherwise the normal paste happens.

Requirements
------------
- macOS (universal arm64 + x86_64 helper) or Windows (x86-64 helper).
- Sublime Text 4 (the package runs under the Python 3.8 plugin host).

Configuration
-------------
Preferences > Package Settings > Markdown Image Paste > Settings

    {
        // "" = same folder as the .md. Example: "assets".
        "image_subdir": ""
    }

Source & issues: https://github.com/vjeantet/sublime-image-paste
